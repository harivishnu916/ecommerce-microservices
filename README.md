# Distributed E-Commerce Microservices

A scalable event-driven e-commerce backend system built using Spring Boot microservices architecture.

## Tech Stack

- Java 17
- Spring Boot
- Spring Cloud
- Apache Kafka
- MySQL
- MongoDB
- Docker
- Kubernetes

## Planned Microservices

- API Gateway
- Order Service
- Inventory Service
- Payment Service
- Notification Service

## Architecture Style

- Microservices Architecture
- Event-Driven Communication
- Saga Pattern for Distributed Transactions

## Team Goal

Build a scalable and resilient backend infrastructure for modern e-commerce applications.
